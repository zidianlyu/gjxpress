## 3. 已修改的文案

### 3.1 首页品牌名称（pages/home/index.wxml）

**修改前**:
```xml
<view class="brand-name">广骏国际快运</view>
```

**修改后**:
```xml
<view class="brand-name">广骏供应链服务</view>
```

**修改原因**: 避免"快运"资质风险，统一使用"供应链服务"定位。

---

### 3.2 首页使用说明步骤4（pages/home/index.wxml）

**修改前**:
```xml
<view class="guide-title">支付与发货</view>
<view class="guide-desc">完成支付后安排国际发货，全程可查询物流</view>
```

**修改后**:
```xml
<view class="guide-title">发货管理</view>
<view class="guide-desc">支付状态确认后安排发货，全程可查询物流状态</view>
```

**修改原因**:
1. 避免"支付"引导词（改为"支付状态确认"仅展示状态）
2. 避免"国际发货"（改为"发货"）
3. 避免"物流"过度承诺（改为"物流状态"强调信息展示）

---

'微信用户标识：用于识别用户身份和订单归属'
