# GJXpress PRD

## Product

广骏供应链服务 provides a Web-based logistics workflow for public customer acquisition, tracking, registration, and internal operations.

## Active Clients

- Public website
- Admin Portal

The retired mini program client is removed from the repository and no longer part of the roadmap.

## Core Capabilities

- Public registration
- Public tracking
- Customer registration review
- Customer and customerCode management
- Inbound package operations
- Customer shipment operations
- Master shipment operations
- Transaction records
- Storage-backed package/customer-shipment images
- Admin authentication and role-based access

## Out Of Scope

- Online payment
- Mini program login or deployment
- Frontend direct Supabase access
- Claims such as tax-included service, guaranteed customs clearance, or guaranteed delivery

## Contact

Public WeChat IDs remain valid contact methods for sales and support copy.
